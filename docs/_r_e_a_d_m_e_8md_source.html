1 # LEAC - Library Evolutionary Algorithms for Clustering <img align="right" width="100" height="100" src="leac_logo.png">
2 
3 Library Evolutionary Algorithms for Clustering (**LEAC**) is a library for the implementation
4 of evolutionary and genetic algorithms to solve the problem of *partition clustering*.
5 The tools is free and open source, under the GNU General Public
6 [GPLv3 license](https://www.gnu.org/licenses/gpl-3.0.en.html)
7 [\[License file\]](../../LICENSE),
8 for Windows&reg;, GNU/Linux&reg; and Mac OS X&reg;.
9 For more details see <a href="https://github.com/kdis-lab/leac/tree/master/leac-userManual.pdf">user
10 manual</a> and <a href="https://hbrobles.github.io/APILeac/index.html">API</a>.
11 Locally the documentation is in the doc folder.
12 
13 Clustering is useful in several exploratory *pattern-analysis*,
14 *grouping*, *decision-making*, and *machine-learning situations*,
15 including *data mining*, *document retrieval*, *image segmentation*,
16 and *pattern classification* <a href="http://doi.acm.org/10.1145/331499.331504">Jain et al</a>.
17 
18 ![](../master/leac_cluster.svg)
19 
20 LEAC is built in C ++ with the current standards of C++11, C++14.
21 Take advantage of the STL library, through the use of
22 data structures and algorithms, for example,
23 Numerics library of Pseudo-random number generation which is
24 fundamental in the genetic and evolutionary algorithms.
25 An additional feature that LEAC has is a low-level software layer
26 based on Streaming SIMD Extensions (SSE) and in the
27 [OpenBLAS](http://www.openblas.net) library ,in order to
28 increase performance. For now only for Linux x86_64, for future
29 versions it is intended to port to other architectures and
30 increase the number of genetic operators implemented with
31 these software layers
32 
33 LEAC also includes several implementations of evolutionary algorithms for
34 partitional clustering which are based on state-of-art of evolutionary
35 proposals in this area:
